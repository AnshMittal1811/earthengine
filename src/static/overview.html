Change Detection with Earth Engine Imagery

A simple Flask web application for performing change detection tasks with Earth Engine imagery.

The AppEngine version (web version) is for demonstration only and has very limited functionality. Exporting results in particular is not possible.

When running on localhost you can carry out the following tasks:

1. Run the iMAD (iteratively re-weighted MAD) algorithm on optical/infrared bitemporal imagery (Landsat 5, 7, 8 and Sentinel-2.)
A. A. Nielsen (2007). The Regularized Iteratively Reweighted MAD Method for Change Detection in Multi- and Hyperspectral Data.
M. J.Canty (2014). Image Analysis, Classification and Change Detection in Remote Sensing, 3rd Ed., CRC Press 2014

2. Run the (sequential) omnibus algorithm on Sentinel-1 polarimetric SAR multitemporal data.
K. Conradsen et al. (2016). Determining the points of change in time series of polarimetric SAR data. IEEE TGRS 54 (5) 3007-3024.

4. Export imagery to your Earth Engine assets folder or to Google Drive for off-line local processing.

Installation

1. Clone this repository or download the ZIP and unpack.
2. Install Anaconda2
3. Install the GEE Python API
4. cd into ../earthengine/src/ and run python app.py
5. Point your browser to localhost:5000

Special thanks to Charles G. Morton for making available his implementation of the incomplete gamma function on the GEE API.
